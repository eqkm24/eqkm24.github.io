사용 횟수 제한 없음. 단, 획득한 총 기회 횟수 범위 내에서만 가능.

방법: 강화 UI 보주 슬롯에 각인서 → 장비 삽입 → 복구 시도
정화의 잎사귀 — 스킬 롤백
획득한 스킬을 바로 전 단계로 롤백합니다.
사용 시 스킬만 삭제 (보주·기회 재지급 없음).
단, 해당 기회는 달인의 각인서로 1회 회복 가능 상태로 변경.
영혼 용해제 — 귀속 해제
